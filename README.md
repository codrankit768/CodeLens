# 🔍 CodeLens AI

CodeLens AI is an AI-powered code analysis tool that helps developers identify issues in their code, understand mistakes, and improve their problem-solving skills.

## ✨ Features

- 🔎 Analyze code for errors and bugs
- 💡 Explain what's wrong and how to fix it
- 🚀 Suggest optimized solutions
- 📊 Display Time & Space Complexity visually
- 🎥 Recommend video tutorials for better understanding
- 📚 Suggest relevant Previous Year Questions (PYQs)
- 🧠 Help prepare for coding interviews

## 🛠️ Tech Stack

- Python
- AI/LLM
- Static Code Analysis
- AST Parsing

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   https://github.com/codrankit768/CodeLens
   ```

2. Navigate to the project directory:
   ```bash
   cd CodeLens-AI
   ```

3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

4. Run the application:
   ```bash
   python main.py
   ```

## 📌 How It Works

1. Paste or upload your code.
2. CodeLens AI analyzes the code.
3. View detected errors and explanations.
4. Check Time & Space Complexity.
5. Watch recommended videos.
6. Practice similar PYQs.
